##来自李一凡的训练建议：

	1）多踢半场攻防和对抗，训练整体性。

	2）每次一个进攻套路

		a.中锋回撤然后分边后卫或者后腰，然后反跑

		b.边后卫出球给边前卫，然后边前卫做给中间的前腰或者回给边后卫
